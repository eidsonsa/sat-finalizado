Projeto da disciplina: Introdução à Computação - IF668

Sat Solver feito em javascript por Eidson Sá (ejas), com base no modelo de Fernando Castor.

Como executar:

No prompt de comando, entre no diretório sat e realize as seguintes linhas de comando:

	var x = require('sat')
	x.solve('nomedoarquivo.cnf>')


O programa irá resolvê-lo, mostrar se o arquivo é SAT e, caso seja, mostrar qual atribuição o torna verdadeiro.
